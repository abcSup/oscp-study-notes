# Proof

## Commands

### Windows

```text
hostname && whoami.exe
type "C:\Documents and Settings\Administrator\Desktop\proof.txt"
ipconfig
```

Search for proof in cmd.exe

```text
C:\> dir /s proof.txt
```

### Linux

proof.txt is located at /root/

```text
hostname && whoami && cat proof.txt && /sbin/ifconfig
```
